📦 Electronics Store Inventory Management System

A complete command-line based inventory and sales management system built using Python and Pandas.
This project allows you to manage stock, track sales, perform analytics, and visualize business insights through graphs.

🚀 Features
🛒 Inventory Management
Add new items to inventory
View complete inventory
Track stock levels
Identify low-stock items

💰 Sales Management
Record sales transactions
Automatically calculate revenue
Maintain structured sales history

🔍 Search System
Search items by:
Brand
Category
Item ID
Advanced search (Mega Search)

📊 Analytics Dashboard
Total sales & profit
Top-selling items and brands
Revenue per item/brand/category
Stock distribution insights
Category vs Brand comparisons

📈 Graph Visualization
Brand vs Sales / Revenue / Stock
Category-based graphs
Monthly sales & revenue trends
Combined “Mega Graphs”
